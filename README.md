# Convert Normal JSON to String Literal JSON

## Problem Statement: 
How to convert a Normal JSON format to a String Literal JSON format.

Normal JSON Format:
```
{
  "id": "101",
  "name": "Ayan Kumar Pan",
  "department": "IT"
}
```

String Literal JSON Format:

```
{\"id\":\"101\",\"name\":\"Ayan Kumar Pan",\"department\":\"IT\"}
```
